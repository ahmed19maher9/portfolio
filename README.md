## 🚀 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmed19maher9/portfolio.git
   cd portfolio
   ```

2. **Install dependencies (optional):**
   ```bash
   npm install
   ```

3. **Preview locally:**
   ```bash
   npm run preview
   ```
   Or simply open `index.html` in your browser

4. **Access the portfolio:**
   Open http://localhost:3000 (with npm preview) or open index.html directly

## 🌐 Deployment on GitHub Pages

### Prerequisites
- GitHub account
- Repository created on GitHub

### Deployment Steps

1. **Create GitHub Repository:**
   - Go to GitHub and create a new repository
   - Name it `portfolio` or `ahmed-maher-portfolio`
   - Make it public

2. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio deployment"
   git branch -M main
   git remote add origin https://github.com/ahmed19maher9/portfolio.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click on "Settings" tab
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"

4. **Access Your Portfolio:**
   Your portfolio will be available at: `https://ahmed19maher9.github.io/portfolio/`

### Automatic Deployment (Recommended)

For easier updates, you can also use GitHub Actions for automatic deployment:

1. Go to your repository Settings → Pages
2. Under "Source", select "GitHub Actions"
3. GitHub will suggest a workflow for static sites - use it
4. Commit and push the workflow file
